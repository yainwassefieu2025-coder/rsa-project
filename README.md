# RSA Project

## Description

This project is about RSA encryption. It shows how to generate keys, encrypt messages, and decrypt them using Python. The goal was to understand how RSA works step by step.

---

## Project Structure

* `RSA_Programming_Project.ipynb` → main file with all the code and explanations

---

## How to Run

1. Make sure you have Python installed
2. Install Jupyter Notebook if needed:
   pip install notebook
3. Run:
   jupyter notebook
4. Open the file `RSA_Programming_Project.ipynb`
5. Run the cells one by one

---

## Dependencies

* Python 3
* Jupyter Notebook

---

## Notes

* This is a simple version of RSA just for learning
* It uses small numbers, so it’s not secure for real use
* Messages are encrypted one character at a time
